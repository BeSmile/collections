# 判断类型
判断对象类型
Object.prototype.toString.apply(param);
输出结果
<pre>
    >>> Object.prototype.toString.apply('')
        "[object String]"
    >>> Object.prototype.toString.apply([])
        "[object Array]"
    >>> Object.prototype.toString.apply(1)
        "[object Number]"
    >>> Object.prototype.toString.apply(null)
        "[object Null]"
    >>> Object.prototype.toString.apply({})
        "[object Object]"
    >>> Object.prototype.toString.apply(undefined)
        "[object Undefined]"
</pre>
